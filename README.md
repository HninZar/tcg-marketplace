# tcg-marketplace

1. React + TypeScript + Next.js

## 1. Installation

### 1.1 Pre-Request

- node version >= 18.6.1
- Next version >= 9.8.0

### 1.2 Setup
```bash
$ npm i

```
### 1.3 Stacks

- CSS => Chakra UI
- Form => Formik

### 1.3 Running the Project

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
